CmdStan

CmdStan is the shell (command-line / terminal) interface to Stan.

CmdStan runs under Windows (DOS, Cygwin), Mac OS X (shell), and Linux (shell).

CmdStan is open source licensed under the 3-clause BSD license (new BSD).

Download Source (with Documentation)

The CmdStan distribution includes the complete source and documentation.

  • CmdStan 2.4.0 (gzipped tar, on GitHub)

After downloading and unpacking, the CmdStan interface guide (including a getting started chapter) and the Stan modeling language manual can be found in the top-level directory doc.

The user's guide for CmdStan can also be downloaded separately:

The guide to installing and getting started with CmdStan on all platforms is currently included with the modeling language manual (see below).
